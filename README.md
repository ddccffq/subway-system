
关于地铁站点未收录的情况：

    本程序的数据来源京港地铁官网和北京本地宝，数据和高德地图上的难免会有偏差，本人已尽可能地遍历地图修改了。同时本程序未收录起始站或终点站是未开通的站点。


关于运行时刻：

    首先网络上的运营时刻各不相同，这里使用的是北京本地宝和京港地铁提供的数据。


关于程序：

    若出现None的站点，原因是Data.xlsx文件sheet=Stations里未录入或者和Distances里的名字不一样

    最开始我是写了不少的用来debug的代码，可能没删干净，但不影响运行

    要求上说可以不考虑换乘站，本程序考虑了换乘站

    输入”计划时间“才能显示”计划的日期“，如果有”计划的日期“必须填”计划时间“，每天”计划时间“默认”计划日期“为今天

    “最短路径”指的就是最短路径，如果经过了未开通站会无法显示，这时可以点“最短最少换乘次数”。

    exe->dist->myprog.exe，点击后可能会等一段时间。

    起点站和终点站是必须填的，如果填错会报错。

    其它标签框选填，若填了“最大换乘次数”再点击“期望最大换成次数下最少时间”会返回一个相对较快的路径。

    “计划出发时间‘的格式例如: 05:54

    计算时间的方式，为了尽可能地考虑到等车的影响，这里在始发站，终点站也做了+60s处理
